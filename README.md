# CursorPublic

## Генератор случайных паролей

Скрипт `password_generator.py` генерирует один или несколько случайных паролей и выводит их в консоль.

### Требования

- Установлен Python 3.x
- Дополнительные зависимости не требуются (используются только стандартные модули `argparse`, `random`, `string`)

### Запуск

Перейдите в корень проекта и выполните:

```bash
python password_generator.py
```

По умолчанию будет сгенерирован один пароль длиной 8 символов, содержащий цифры, буквы и специальные символы.

### Доступные аргументы

- `-l, --length` — длина пароля (по умолчанию `8`).
- `-n, --number` — количество паролей (по умолчанию `1`).
- `--no-digits` — не включать цифры.
- `--no-letters` — не включать буквы.
- `--no-symbols` — не включать специальные символы.
- `--word` — слово, которое будет включено в пароль и предварительно преобразовано в leet-стиль.

### Примеры

Сгенерировать один пароль длиной 12 символов:

```bash
python password_generator.py -l 12
```

Сгенерировать 5 паролей длиной 16 символов только из букв и цифр:

```bash
python password_generator.py -l 16 -n 5 --no-symbols
```

Сгенерировать 3 пароля длиной 10 символов только из цифр:

```bash
python password_generator.py -l 10 -n 3 --no-letters --no-symbols
```

Сгенерировать пароль длиной 12 символов с включённым словом (в leet-стиле):

```bash
python password_generator.py -l 12 --word example
```
